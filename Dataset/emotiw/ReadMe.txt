Group Affect Database 3.0

This data is part of the Emotion Recognition in the Wild challenge 2018. Note that the database is provided as is. The images are owned by their respective owners. We donot own any of the images.
The data can only be used for academic research.

This task being a three class problem, the data has been organised into zip files. The name of a .zip file indicates the class of an image inside the zip file.
